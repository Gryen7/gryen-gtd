LaravelBlog
====

基于 laravel 的 Blog 应用，提供以下功能：
----

- 博客写作
- 任务列表

版本更新预告
----

#### v0.0.8

- 首页 “听” 优化
- 暂存文章不跳转，直接在当前页面给出提示
- 首页随笔板块优化
- 首页推荐默认按照推荐时间倒序排列
