# README

gryen-gtd 是一个界面简洁的 web 端个人 gtd app。基于 Laravel 开发，提供博客发布、任务列表等功能。

### 技术一览
* 基于 Laravel 5.6，需要 php 7.1.3 以上版本
* 推荐使用 MariaDB 数据库
* 推荐使用 Redis 缓存数据库
